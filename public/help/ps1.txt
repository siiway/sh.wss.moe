Custom PS1 Prompt Setup

Usage:
  bash <(curl -fsS https://sh.wss.moe/ps1)
  bash <(curl -fsS https://sh.wss.moe/ps1) --git
  bash <(curl -fsS https://sh.wss.moe/ps1) --no-newline
  bash <(curl -fsS https://sh.wss.moe/ps1) --git --no-newline

Features:
  Default: Multi-line prompt without Git status
  --git: Include Git branch and simple status indicators
  --no-newline: Single-line prompt
  Appends to ~/.bashrc (with backup of original)
  Colors: time(green), user(yellow), @ (purple), path(red), git(blue)

No sudo required.
