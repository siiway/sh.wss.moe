Custom PS1 Prompt Setup

Usage:
  curl -fsSL sh.wss.moe/ps1 | bash
  bash <(curl -fsSL sh.wss.moe/ps1)
  curl -fsSL sh.wss.moe/ps1 | bash -s -- --git
  bash <(curl -fsSL sh.wss.moe/ps1) --git
  curl -fsSL sh.wss.moe/ps1 | bash -s -- --no-newline
  bash <(curl -fsSL sh.wss.moe/ps1) --no-newline
  curl -fsSL sh.wss.moe/ps1 | bash -s -- --git --no-newline
  bash <(curl -fsSL sh.wss.moe/ps1) --git --no-newline

Features:
  Default: Multi-line prompt without Git status
  --git: Include Git branch and simple status indicators
  --no-newline: Single-line prompt
  Writes prompt config to ~/.wssmoe/ps1.env
  Adds one loader line to ~/.bashrc if missing
  Colors: time(green), user(yellow), @ (purple), path(red), git(blue)

No sudo required.
