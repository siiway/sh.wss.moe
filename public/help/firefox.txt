Firefox from Mozilla PPA

Usage:
  curl https://sh.wss.moe/firefox | sudo bash
  curl https://sh.wss.moe/firefox | sudo bash -s -- --no-gnome   # skip GNOME integration

Features:
  Adds PPA only if missing
  Sets high-priority pinning
  Installs latest Firefox
  GNOME support optional (--no-gnome to skip)

Requires: sudo
From: https://launchpad.net/~mozillateam/+archive/ubuntu/ppa
