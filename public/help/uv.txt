uv (Fast Python Tool) Installation

Usage:
  curl -fsSL sh.wss.moe/uv | bash
  bash <(curl -fsSL sh.wss.moe/uv)
  curl -fsSL sh.wss.moe/uv | bash -s -- 3.13     # also install Python 3.13
  bash <(curl -fsSL sh.wss.moe/uv) 3.13     # also install Python 3.13
  curl -fsSL sh.wss.moe/uv | bash -s -- 3.12     # install Python 3.12, etc.
  bash <(curl -fsSL sh.wss.moe/uv) 3.12     # install Python 3.12, etc.

Features:
  - Installs uv via official script
  - Default: Only installs uv, no Python version
  - Pass a version (e.g. 3.13) as first argument to also install that Python version
  - After install, run: source ~/.bashrc

No sudo required
Pre-dependency: curl
Ref: https://wyf9.top/p/linux-newinst#python-uv
