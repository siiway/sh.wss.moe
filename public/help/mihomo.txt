mihomo Installer (Linux)

Usage:
  curl -fsSL sh.wss.moe/mihomo | sudo bash
  curl -fsSL sh.wss.moe/mihomo | sudo bash -s -- arm64-v8 --alpha
  curl -fsSL sh.wss.moe/mihomo | sudo bash -s -- --no-mirror
  sudo bash <(curl -fsSL sh.wss.moe/mihomo)
  sudo bash <(curl -fsSL sh.wss.moe/mihomo) amd64 --alpha
  sudo bash <(curl -fsSL sh.wss.moe/mihomo) --no-mirror

Features / Notes:
  - Downloads Linux release binaries from MetaCubeX/mihomo
  - Reads `version.txt` from the release first, then builds the download URL directly
  - Latest stable uses the `compatible` build from the latest stable tag
  - Use `--alpha` to install `Prerelease-Alpha`
  - Auto-detects arch by default: amd64, 386, arm64-v8, armv7
  - You can override arch with arg1, such as `amd64` or `arm64-v8`
  - Installs binary to /usr/local/bin/mihomo
  - Mirror enabled by default, use --no-mirror to disable

Requires: sudo
