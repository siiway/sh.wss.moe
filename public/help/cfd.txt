Cloudflared (official APT repo)

Usage:
  curl -fsSL sh.wss.moe/cfd | sudo bash
  sudo bash <(curl -fsSL sh.wss.moe/cfd)
  curl -fsSL sh.wss.moe/cfd | sudo bash -s -- --skip-fix   # skip sysctl fixes
  sudo bash <(curl -fsSL sh.wss.moe/cfd) --skip-fix   # skip sysctl fixes

Features:
  Installs from Cloudflare repo
  Applies buffer (if config exists) + ping fixes by default
  Shows version after install

Requires: sudo
Pre-dependency: curl
From: https://pkg.cloudflare.com/
Ref: https://wyf9.top/p/linux-newinst#cloudflared
