Change Linux Hostname

Usage:
  curl https://sh.wss.moe/hostname | sudo bash new-hostname
  curl https://sh.wss.moe/hostname | sudo bash new-hostname --temp   # temporary only (reboot reverts)

Examples:
  curl https://sh.wss.moe/hostname | sudo bash my-server-01
  curl https://sh.wss.moe/hostname | sudo bash cloud-node --temp

Features:
  - Default: Permanent change (updates /etc/hostname and /etc/hosts)
  - --temp: Temporary change (current session only)
  - Automatic backup of original files
  - Basic validation (alphanumeric + hyphen)
  - Immediate apply for new hostname

Requires: sudo
