Change Linux Hostname

Usage:
  curl -fsSL sh.wss.moe/hostname | sudo bash -s -- new-hostname
  sudo bash <(curl -fsSL sh.wss.moe/hostname) new-hostname
  curl -fsSL sh.wss.moe/hostname | sudo bash -s -- new-hostname --temp   # temporary only (reboot reverts)
  sudo bash <(curl -fsSL sh.wss.moe/hostname) new-hostname --temp   # temporary only (reboot reverts)

Examples:
  sudo bash <(curl -fsSL sh.wss.moe/hostname) my-server-01
  sudo bash <(curl -fsSL sh.wss.moe/hostname) cloud-node --temp

Features:
  - Default: Permanent change (updates /etc/hostname and /etc/hosts)
  - --temp: Temporary change (current session only)
  - Automatic backup of original files
  - Basic validation (alphanumeric + hyphen)
  - Immediate apply for new hostname

Requires: sudo
