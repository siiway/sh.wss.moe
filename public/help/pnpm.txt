pnpm Package Manager

Usage:
  curl https://sh.wss.moe/pnpm | bash
  curl https://sh.wss.moe/pnpm | bash -s -- --no-mirror   # skip China mirror

Features:
  Installs pnpm in user scope via npm
  Sets npmmirror registry by default (fast in China)
  Use --no-mirror to skip mirror

No sudo required
Pre-dependency: npm (from node)
From: https://pnpm.io/installation#using-npm
