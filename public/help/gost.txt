gost Installer (Linux)

Usage:
  curl -fsSL sh.wss.moe/gost | sudo bash
  sudo bash <(curl -fsSL sh.wss.moe/gost)
  curl -fsSL sh.wss.moe/gost | sudo bash -s -- amd64 --no-verify
  sudo bash <(curl -fsSL sh.wss.moe/gost) amd64 --no-verify

Features / Notes:
  - Fetches latest stable release tag from GitHub API
  - Linux only
  - Auto-detects architecture by default (arg1=auto)
  - You can override arch with arg1, such as amd64/arm64/armv7/mipsle_softfloat
  - Verifies SHA256 using release checksums.txt by default (arg2=--verify)
  - Use --no-verify to skip checksum verification
  - Installs binary to /usr/local/bin/gost

Requires: sudo
