mmproxy

Usage:
  curl -fsSL sh.wss.moe/mmproxy | bash
  curl -fsSL sh.wss.moe/mmproxy | bash -s -- --no-mirror
  bash <(curl -fsSL sh.wss.moe/mmproxy)
  bash <(curl -fsSL sh.wss.moe/mmproxy) --no-mirror

Features:
  Downloads pre-built mmproxy binary from GitHub releases
  Installs to /usr/local/bin/mmproxy
  mmproxy converts PROXY protocol to real source IP TCP connections
  Mirror enabled by default, use --no-mirror to disable (mirror: gh.1s.fan)

Requires: sudo
Pre-dependency: curl
From: https://github.com/cloudflare/mmproxy (Source Code)
      https://github.com/SiiWay/mmproxy (Binary)
