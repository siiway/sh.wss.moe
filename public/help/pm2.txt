PM2 Process Manager

Usage:
  curl -fsSL sh.wss.moe/pm2 | bash
  bash <(curl -fsSL sh.wss.moe/pm2)

Features:
  Installs PM2 (user-scope) via bun/pnpm/npm (tries in order)

No sudo required for install
Pre-dependency: bun, pnpm, or npm
From: https://pm2.keymetrics.io/docs/usage/quick-start/
Ref: https://wyf9.top/p/linux-newinst#pm2
