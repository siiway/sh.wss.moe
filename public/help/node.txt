Node.js via nvm

Usage:
  bash <(curl -fsS https://sh.wss.moe/node)
  bash <(curl -fsS https://sh.wss.moe/node) 20   # custom version

Features:
  Installs nvm and specified Node version (default 25)
  Loads nvm in current script run (no manual source needed)

No sudo required
Pre-dependency: curl
From: https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating
