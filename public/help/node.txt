Node.js via nvm

Usage:
  curl https://sh.wss.moe/node | bash
  curl https://sh.wss.moe/node | bash -s -- 20   # custom version

Features:
  Installs nvm and specified Node version (default 25)

No sudo required
Pre-dependency: curl
