Node.js via nvm

Usage:
  curl -fsSL sh.wss.moe/node | bash
  bash <(curl -fsSL sh.wss.moe/node)
  curl -fsSL sh.wss.moe/node | bash -s -- 20   # custom version
  bash <(curl -fsSL sh.wss.moe/node) 20   # custom version

Features:
  Installs nvm and specified Node version (default 25)
  After install, run: source ~/.bashrc

No sudo required
Pre-dependency: curl
From: https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating
