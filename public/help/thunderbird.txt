Thunderbird from Mozilla PPA

Usage:
  sudo bash <(curl -fsS https://sh.wss.moe/thunderbird)
  sudo bash <(curl -fsS https://sh.wss.moe/thunderbird) --no-gnome   # skip GNOME integration

Features:
  Adds PPA only if missing
  Sets high-priority pinning
  Installs Thunderbird
  GNOME support optional (--no-gnome to skip)

Requires: sudo
From: https://launchpad.net/~mozillateam/+archive/ubuntu/ppa
