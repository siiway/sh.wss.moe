Waylyrics (Wayland Lyrics Display)

Usage:
  curl -fsSL sh.wss.moe/waylyrics | sudo bash
  curl -fsSL sh.wss.moe/waylyrics | sudo bash -s -- --no-mirror
  sudo bash <(curl -fsSL sh.wss.moe/waylyrics)
  sudo bash <(curl -fsSL sh.wss.moe/waylyrics) --no-mirror

Features:
  Clones repo, builds with cargo, installs to system paths
  Includes zh_CN translation and themes
  Mirror enabled by default, use --no-mirror to disable (mirror: gh.1s.fan)

Requires: sudo
From: https://github.com/waylyrics/waylyrics/blob/master/doc/INSTALLATION.md
Ref: https://wyf9.top/p/linux-newinst#waylyrics
